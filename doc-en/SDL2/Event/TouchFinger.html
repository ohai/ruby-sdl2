Overview

This class represents touch finger events.

You don't handle the instance of this class directly, but you handle the instances of two subclasses of this subclasses: FingerMotion and MouseButtonUp.

Direct Known Subclasses

FingerDown, FingerMotion, FingerUp

static VALUE EvTouchFinger_inspect(VALUE self)
{
    SDL_Event* ev; Data_Get_Struct(self, SDL_Event, ev);
    return rb_sprintf("<%s: type=%u timestamp=%u"
                      " touch_id=%d finger_id=%d"
                      " x=%f y=%f pressure=%f>",
                      rb_obj_classname(self), ev->common.type, ev->common.timestamp,
                      (int)ev->tfinger.touchId, (int)ev->tfinger.fingerId,
                      ev->tfinger.x, ev->tfinger.y, ev->tfinger.pressure);
}
